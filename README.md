# helloMVC

MVC 디자인 패턴에 입각하여 동적인 웹 애플리케이션 만들기

- Servlet과 JSP 사용

- 메모리상에 임의의 Customer 객체를 생성하여 HashMap에 저장하고 id를 바탕으로 Customer를 조회하는 서비스
